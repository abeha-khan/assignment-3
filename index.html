How to practice (quick)

  1. Fork the repo on GitHub.
  2. Clone your fork and create a branch: git checkout -b feature-about-edit.
  3. Edit this file and commit: git commit -am "Improve about copy".
  4. Push and open a Pull Request from your branch.
